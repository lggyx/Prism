## 1. 接口总览

### 1.1 Base URL

```text
https://api.prism.example.com/api/v1
```

本地开发可使用：

```text
http://localhost:3000/api/v1
```

### 1.2 认证方式

MVP 建议匿名设备身份 + 可选登录账号。

```http
Authorization: Bearer <access_token>
X-Device-Id: <device_uuid>
```

若用户未登录，后端仍应基于 `X-Device-Id` 创建匿名观察者编号，例如 `OBS-0042`。

### 1.3 通用响应结构

成功：

```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

失败：

```json
{
  "code": "PRISM_VALIDATION_ERROR",
  "message": "lensId is required",
  "requestId": "req_20260516_xxx"
}
```

### 1.4 分页参数

```text
limit: number, 默认 20, 最大 50
cursor: string, 游标分页
```

分页响应：

```json
{
  "items": [],
  "nextCursor": "cur_xxx",
  "hasMore": true
}
```

## 2. 核心数据模型

### 2.1 Lens 镜片

```ts
type Lens = {
  id: string;                 // naturalist
  name: string;               // 博物学家
  englishName: string;        // NATURALIST
  category: "NATURE" | "CULTURE" | "URBAN" | "TEMPORAL";
  color: string;              // #5DCAA5
  icon: string;               // ti-leaf
  description: string;
  fullDescription?: string;
  usageCount: number;         // 当前用户使用次数
  globalUsageCount?: number;  // 社区使用次数
  isPreset: boolean;
  isAvailable: boolean;
};
```

### 2.2 CaptureAsset 拍摄素材

```ts
type CaptureAsset = {
  id: string;
  imageUrl: string;
  width: number;
  height: number;
  mimeType: "image/jpeg" | "image/png" | "image/webp";
  exif?: {
    iso?: number;
    shutter?: string;
    aperture?: string;
  };
  location?: GeoLocation;
  capturedAt: string;
};
```

### 2.3 GeoLocation 位置

```ts
type GeoLocation = {
  latitude: number;
  longitude: number;
  accuracyMeters?: number;
  city?: string;
  province?: string;
  displayPrivate: string; // "31.2304°N 121.4737°E"
  displayPublic: string;  // "上海 · 上海" 或 "31.2°N 121.5°E"
};
```

隐私规则：

| 场景 | 字段 |
|---|---|
| 用户自己收藏页 | `displayPrivate`，保留精确坐标 |
| 发现页社区流 | `displayPublic`，城市级或一位小数坐标 |
| 用户关闭社区位置展示 | `displayPublic = null` |

### 2.4 Annotation AI 标注

```ts
type Annotation = {
  id: string;
  text: string;
  target: {
    x: number; // 0-1，相对图片宽度
    y: number; // 0-1，相对图片高度
  };
  labelBox?: {
    x: number;
    y: number;
    width: number;
  };
  line?: Array<{ x: number; y: number }>;
  confidence?: number;
};
```

### 2.5 Reading 重述结果

```ts
type Reading = {
  id: string;
  captureId: string;
  lens: Lens;
  imageUrl: string;
  summary: string;
  annotations: Annotation[];
  status: "processing" | "succeeded" | "failed";
  createdAt: string;
};
```

### 2.6 Slice 收藏切片

```ts
type Slice = {
  id: string;
  readingId: string;
  captureId: string;
  lens: Lens;
  imageUrl: string;
  thumbnailUrl: string;
  summary: string;
  annotations: Annotation[];
  location?: GeoLocation;
  isPublic: boolean;
  resonanceCount: number;
  hasResonated?: boolean;
  savedAt: string;
  createdAt: string;
};
```

## 3. 取景与拍摄接口

### 3.1 上传拍摄照片

取景页按下快门后调用。前端完成相机采集，后端负责存储原图并返回 `captureId`。

```http
POST /captures
Content-Type: multipart/form-data
```

请求字段：

| 字段 | 类型 | 必填 | 说明 |
|---|---:|---:|---|
| image | File | 是 | 拍摄照片 |
| capturedAt | string | 是 | ISO 时间 |
| latitude | number | 否 | 纬度 |
| longitude | number | 否 | 经度 |
| accuracyMeters | number | 否 | 定位精度 |
| iso | number | 否 | 装饰/EXIF 信息 |
| shutter | string | 否 | 如 `1/120` |
| aperture | string | 否 | 如 `f/2.0` |

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "id": "cap_01HZX",
    "imageUrl": "https://cdn.example.com/captures/cap_01HZX.jpg",
    "width": 3024,
    "height": 4032,
    "mimeType": "image/jpeg",
    "location": {
      "latitude": 31.2304,
      "longitude": 121.4737,
      "accuracyMeters": 18,
      "city": "上海",
      "province": "上海",
      "displayPrivate": "31.2304°N 121.4737°E",
      "displayPublic": "上海 · 上海"
    },
    "capturedAt": "2026-05-16T10:12:30+08:00"
  }
}
```

## 5. AI 重述接口

### 5.1 创建重述任务

镜片选择页点击镜片后调用。后端调用多模态模型识别画面中存在的项目，并按镜片世界观生成标注和总结。

```http
POST /readings
Content-Type: application/json
```

请求：

```json
{
  "captureId": "cap_01HZX",
  "lensId": "naturalist",
  "language": "zh-CN"
}
```

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "readingId": "read_01HZY",
    "status": "processing"
  }
}
```

### 5.2 查询重述结果

前端扫描加载期间轮询，建议 500-1000ms 一次，最多 30s。

```http
GET /readings/{readingId}
```

成功响应：

```json
{
```

### 6.4 删除切片

收藏页长按操作面板 `DELETE`。

```http
DELETE /slices/{sliceId}
```

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "deleted": true
  }
}
```

### 6.5 重新解析切片

收藏页长按操作面板 `REANALYZE`。可沿用原图，选择同一镜片或新镜片。

```http
POST /slices/{sliceId}/reanalyze
Content-Type: application/json
```

请求：

```json
{
  "lensId": "ruin-archaeology"
}
```

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "readingId": "read_02AAA",
    "status": "processing"
  }
}
```

### 6.6 导出切片

收藏页长按操作面板 `EXPORT`。后端生成带照片、标注、总结语的分享图。

```http
POST /slices/{sliceId}/export
Content-Type: application/json
```

请求：

```json
{
  "format": "image/png",
  "includeLocation": false
}
```

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "exportUrl": "https://cdn.example.com/exports/slice_01HZQ.png",
    "expiresAt": "2026-05-17T10:20:00+08:00"
  }
}
```

### 8.2 更新个人设置

```http
PATCH /me/settings
Content-Type: application/json
```

请求：

```json
{
  "showCommunityLocation": true,
  "locationPrecision": "DISTRICT",
  "challengeNotifications": false,
  "interfaceTheme": "DARK",
  "defaultSlicePublic": false
}
```

所有字段均可选，前端只需提交发生变化的字段。

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "showCommunityLocation": true,
    "locationPrecision": "DISTRICT",
    "challengeNotifications": false,
    "interfaceTheme": "DARK",
    "defaultSlicePublic": false
  }
}
```

约束：

1. 当 `showCommunityLocation = false` 时，后端应自动将社区流中的该用户位置展示置空。
2. 当 `locationPrecision = OFF` 时，即使 `showCommunityLocation = true`，社区流也不展示位置。
3. 当前 MVP 只支持 `interfaceTheme = DARK`，其他值返回 `PRISM_VALIDATION_ERROR` 或忽略并返回 `DARK`。

### 8.3 退出登录

个人页点击 `Sign out` 并确认后调用。匿名设备身份可只清除当前 access token，保留本地 `X-Device-Id` 对应的匿名观察者数据。

```http
POST /auth/signout
```

响应：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "signedOut": true
  }
}
```

## 12. AI 重述生成约束

后端生成 `annotations` 时应遵守产品规范：

1. 不强制识别“主体”，只标注画面中可见项目。
2. 默认生成 5 条标注，MVP 可允许 3-5 条。
3. 标注文字最多两行，单条建议 16-36 个中文字符。
4. 标注坐标使用 0-1 相对坐标，避免前端不同图片比例下失准。
5. 总结语要体现镜片世界观，不写成通用图片描述。
6. 发现页只返回公开切片，且必须使用脱敏位置。
