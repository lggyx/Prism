## 一、产品是什么

**世界观透镜**是一款移动 App（React + Capacitor Android），核心体验是：用户拍一张照片，选择一个"世界观镜片"（如博物学家、宋代文人、废墟考古等），AI 会用该视角重新解读照片内容，以田野笔记式的标注线+文字散落在照片上呈现。每次解读保存为一枚"认知切片"（照片+标注+世界观+时空坐标），积累成用户的个人视角档案。

**一句话定位**：不是帮用户识别世界的工具，而是帮用户收藏视角、扩展视角、重新理解世界的认知透镜。

**产品核心流程**：拍下现实 → 选择镜片 → 世界重述（AI 标注）→ 保存视角

## 二、已完成的设计工作

### 2.1 全局视觉规范（已确认）

- **设计语言**："高科技光学仪器"——深色背景 #0a0c10、等宽字体、小圆角 4-6px、细线条、白色不同 opacity 层级
- **避免社媒感**：不用大圆角、不用药丸按钮、不用瀑布流、用竖线分隔数据、用刻度盘风格筛选
- **镜片颜色系统**：博物学家绿 #5DCAA5 / 都市倦怠灰 #B4B2A9 / 宋代文人琥珀 #FAC775 / 废墟考古珊瑚 #F0997B / 经济学家蓝 #85B7EB / 儿童视角粉 #ED93B1
- **Tab Bar**：仪器面板风格，5 个 Tab：COLLECT | LENSES | CAPTURE（中央）| DISCOVER | PROFILE

### 2.2 七个页面的 UI 交互方案（全部已确认 + 已出 Codex Prompt + 已跑出 HTML 原型）

### 2.3 已确认的关键设计决策

| 决策 | 结论 |
|------|------|
| 拍照与选镜片 | 两步分离，但在同一页面内连贯过渡（A iris close → 照片上移 → B 镜片列表弹出） |
| 图片来源 | 仅实时拍照，不接入相册 |
| 镜片选择模式 | MVP 单选 + 预设镜片，UI 预留多选和自定义扩展位 |
| AI 解读方式 | 不识别"主体"，仅识别画面中存在的项目，标注散落在照片各处 |
| 标注文字呈现 | 逐字打出（打字机效果） |
| 保存切片动效 | 模拟手机截屏直觉——画面闪白 |
| 自定义镜片创建 | 通过 AI 语音对话（STT + LLM），从镜片选择页底部入口触发 |
| 社区身份 | MVP 用 OBS-XXXX 自动编号（匿名），后续升级为自选观察者代号 |
| 社区互动语言 | ◇ 共鸣（非❤️点赞），OBS 编号（非头像昵称），#07 期数编号 |
| 位置隐私 | 私人数据保留精确坐标；社区数据模糊到城市级别；用户可设置 EXACT/CITY/HIDDEN 三档 |
| 收藏页风格 | 高科技底片浏览器（非瀑布流社媒），统一 3:4 网格 |
