## 全局视觉规范

### 设计语言："高科技光学仪器"

所有页面统一遵循以下风格：

| 维度 | 规范 | 避免 |
|------|------|------|
| 背景色 | #0a0c10 深色沉浸 | 白色/浅色背景 |
| 圆角 | 4-6px（精密感） | 大圆角（社交 App 感） |
| 字体 | 等宽字体用于元数据、标签、导航 | 全部用无衬线体 |
| 颜色策略 | 白色的不同 opacity 层级（0.06/0.15/0.25/0.35/0.5/0.7/0.9），彩色仅用于镜片标识色点 | 大面积彩色背景 |
| 图标 | 线条式（Tabler outline） | 实心填充图标 |
| 分割线 | 1px，rgba(255,255,255,0.04-0.06) | 粗分割线 |
| 信息展示 | 竖线分隔数据、仪器面板风格 | 药丸按钮、大标题 |

### 等宽字体栈

```
'SF Mono', 'Menlo', 'Courier New', monospace
```

### 镜片颜色标识（全局统一）

| 镜片 | 标识色 | 用途 |
|------|--------|------|
| 博物学家 | 绿色 #5DCAA5 | 色点、边框、光晕 |
| 都市倦怠 | 灰色 #B4B2A9 | 同上 |
| 宋代文人 | 琥珀色 #FAC775 | 同上 |
| 废墟考古 | 珊瑚色 #F0997B | 同上 |
| 经济学家 | 蓝色 #85B7EB | 同上 |
| 儿童视角 | 粉色 #ED93B1 | 同上 |

### 底部 Tab Bar（全局统一）

- 风格：仪器面板，非常规 Tab Bar
- 背景：纯深色 #0d0f14，顶部极细分割线
- 5 个 Tab：COLLECT | LENSES | CAPTURE（中央突出）| DISCOVER | PROFILE
- 图标：线条式，不实心填充
- 文字：等宽字体 9px
- 选中态：白色图标 + 白色文字 + 上方 2px 短指示线
- 未选中：rgba(255,255,255,0.25)
- CAPTURE 按钮：带细圆环的方形/六边形按钮，视觉突出但不凸起

## Page 1: 取景页（Viewfinder）

### 定位

观察的起点。不是"拍照"，是"举起一个观察仪器"。

### 图片来源

仅支持实时相机拍摄，不接入相册。

### 页面元素

| 区域 | 内容 | 说明 |
|------|------|------|
| 背景层 | 相机实时取景画面，全屏 | — |
| 仪器感叠层 | 同心圆准星 + 十字线（opacity 极低） | 精密但安静，类似徕卡取景器气质 |
| 左下角 | GPS 坐标 + 日期 | 等宽字体，半透明，胶片数据背印风格 |
| 右下角 | 模拟光圈 / ISO / 快门参数 | 装饰性信息，强化仪器感 |
| 左上角 | 关闭按钮（×） | 半透明圆形背景 |
| 右上角 | 闪光灯切换 | 半透明圆形背景 |
| 底部居中 | 快门按钮 | 唯一核心操作，双圈设计 |

### 设计原则

- 极简：仅 3 个可交互元素（关闭、闪光灯、快门）
- 无滤镜、无变焦、无倒计时、无模式切换

### 过渡动效：Iris Close（已确认）

按下快门后（约 1.2 秒）：

1. **定格**（0ms）：取景画面停止，快门按钮消失
2. **Iris 收拢**（0-600ms）：圆形遮罩从四周向中心收缩（clip-path circle 100%→35%），外部变深色半透明
3. **画面后退**（400-800ms）：scale 1.0→0.92，blur 0→2px
4. **镜片选项浮现**（600-1200ms）：从底部滑入，opacity 0→1，translateY 40px→0

## Page 2: 镜片选择页（Lens Picker）

### 定位

用户刚拍完照片，决定"用谁的眼睛来重新看它"。

### MVP 策略

- 当前：单选 + 预设镜片（点击即跳转）
- UI 预留：多选对比 + 自定义镜片入口

### 页面布局

| 区域 | 内容 | 说明 |
|------|------|------|
| 顶部导航 | 左：返回 / 中："选择镜片"（等宽小字）/ 右：留空 | — |
| 上方 30% | 照片缩略图（3:4，圆角） | 周围淡圆形虚线，模拟"透镜框住样本" |
| 下方 60% | 镜片卡片列表（可滚动） | 小标题"预设镜片" |
| 列表底部 | "+ 创建自定义镜片"（虚线卡片） | "即将推出"，视觉降权 |

### 预设镜片（MVP 4 个）

| 镜片 | 色彩 | 描述 |
|------|------|------|
| 博物学家 | 绿色 | 生态标本与田野笔记 |
| 都市倦怠 | 灰色 | 疲惫都市人的凝视 |
| 宋代文人 | 琥珀色 | 清供雅趣与闲适心境 |
| 废墟考古 | 珊瑚色 | 当下之物的未来化石 |

### 交互

- 点击镜片卡片 → 高亮 → 列表下滑消失 → 照片放大全屏 → 扫描加载 → 进入重述结果页
- 点击"创建自定义" → toast "即将推出"

## Page 3: 重述结果页（Restatement）

### 定位

产品的"啊哈时刻"。AI 用选定世界观重新解读照片。

### 核心体验

照片全屏打底，AI 解读以"标注线 + 文字"散落在照片各处，像科学田野笔记。标注逐条出现，文字逐字打出（打字机效果）。

### 关键理念

**不识别"主体"，仅识别画面中存在的项目。** 不需要判断核心主体，只要画面里有东西就能标注。容错率极高。

### 页面布局

| 区域 | 内容 | 说明 |
|------|------|------|
| 背景层 | 用户照片全屏 + 极淡暗色蒙版 | 确保白色标注可读 |
| 顶部导航 | 左：返回 / 右：镜片标签（药丸状） | 悬浮，半透明背景 |
| 标注层 | 5 个标注点（圆点 + 线 + 文字） | 分散在画面不同位置 |
| 底部操作栏 | 左："换镜片" / 中：总结语 / 右："保存切片" | 半透明毛玻璃背景 |

### 标注结构（每条 = 3 部分）

1. 标注圆点（6px）— 物体位置
2. 标注线（1px 白色半透明）— 斜线或折线，60-100px
3. 标注文字（12-13px 白色）— 线末端，最多两行

### 动画序列

| 阶段 | 时间 | 内容 |
|------|------|------|
| 扫描感 | 0-500ms | 白色横线从顶扫到底，蒙版提亮 |
| 标注出现 | 500-4000ms | 逐条出现（圆点弹出→线生长→文字打字机），每条间隔 700ms |
| 总结语 | 4000-5500ms | 底部逐字打出总结语 |

### 交互

| 操作 | 行为 |
|------|------|
| 点击标注 | 高亮放大，其余降透明度，3s 恢复 |
| 换镜片 | 标注反序收回 → 返回镜片选择（同一张照片） |
| 保存切片 | 标注闪烁 → 画面闪白（截屏感）→ toast → 按钮变灰 |

## Page 4: 收藏页（Collection）

### 定位

用户所有认知切片的沉淀。高科技底片浏览器，不是社交 feed。

### 页面布局

| 区域 | 内容 | 说明 |
|------|------|------|
| 顶部 | "COLLECTION" + "12 SLICES"（等宽小字） | 仪器面板风格 |
| 筛选条 | ALL \| 博物学家 \| 都市倦怠 \| 宋代文人 \| 废墟考古 | 刻度盘风格，竖线分隔，非药丸按钮 |
| 内容区 | 2 列统一网格（非瀑布流），固定 3:4 比例 | 底片浏览器感 |
| 底部 | Tab Bar | COLLECT 选中 |

### 切片卡片结构（底片风格）

| 层级 | 内容 |
|------|------|
| 底片边框 | 左右两侧各 4 个等距小方点（3px，极淡），模拟胶片齿孔 |
| 照片区域 | 暗色调，圆角 4px |
| 底部元数据条 | 镜片色点 + 名称（等宽 10px）+ 总结语（11px，一行截断） |
| 左上角 | 日期（等宽 9px，如 "05.16"） |
| 右上角 | GPS 标记（4px 圆点） |

### 交互

- 筛选：指示线滑动 + 内容 fade
- 点击卡片：边框变亮 → 全屏展开 → 详情视图（照片 + 标注 + 完整信息）
- 长按卡片：底部操作面板（REANALYZE / EXPORT / DELETE）

## Page 6: 发现页（Discover）

### 定位

"扩展视角"的落地。信号接收面板，不是社交 feed。

### 页面布局（整页可滚动）

| 区域 | 内容 | 说明 |
|------|------|------|
| 顶部 | "DISCOVER" + "1,247 OBSERVERS" | 仪器面板风格 |
| 模块一 | 视角挑战（WEEKLY CHALLENGE） | 单张横幅卡片 |
| 模块二 | 热门镜片（TRENDING LENSES） | 横向可滚动，4 个小卡片 |
| 模块三 | 社区切片流（SIGNAL FEED） | 单列纵向列表 |
| 底部 | Tab Bar | DISCOVER 选中 |

### 视角挑战卡片

| 位置 | 内容 |
|------|------|
| 左侧 | 期数编号 "#07"（大号等宽数字） |
| 中间 | 挑战标题 + 描述 + "38 JOINED \| 3D LEFT" |
| 右侧 | "JOIN" 按钮 |
| 底部 | 时间进度条（镜片颜色，1px） |

### 热门镜片（横向滚动）

每个小卡片（90×100px）：镜片图标 + 名称 + 热度（"↑ 142"）

### 社区切片流（SIGNAL FEED）

每个条目结构：

| 位置 | 内容 |
|------|------|
| 头部行 | 观察者编号 "OBS-0042"（左）+ 时间戳 "2h ago"（右） |
| 照片区 | 模拟照片（180px）+ 简化标注线 + 左下角镜片标签 |
| 总结语 | 一行文字 |
| 底部数据 | 模糊坐标 + "◇ 12"（共鸣数） |

### 社区交互语言（避免社媒感）

| 常规社媒 | 本产品替代方案 | 原因 |
|----------|--------------|------|
| 用户头像 + 昵称 | OBS-XXXX 编号 | 焦点在内容不在人，仪器感 |
| ❤️ 点赞 | ◇ 共鸣 | 菱形更几何更仪器，"共鸣"比"喜欢"更认知 |
| "第七期" | "#07" | 仪器面板节奏感 |
| 精确坐标 | 城市级模糊 | 隐私保护（见下方隐私策略） |

## 隐私与社区身份策略

### 位置信息分级

| 场景 | 坐标精度 | 说明 |
|------|----------|------|
| 用户自己的收藏页 | 精确坐标（如 36.6712°N 117.0515°E） | 私人数据，完整保留 |
| 社区切片流（发现页） | 城市级模糊（如 "济南 · 山东" 或一位小数 36.7°N 117.1°E） | 约 10km 精度，防止定位到具体位置 |
| 用户设置 | 可选择关闭社区位置展示 | 尊重用户选择 |

### 社区身份

| 阶段 | 方案 | 说明 |
|------|------|------|
| MVP | OBS-XXXX 自动编号 | 完全匿名，焦点在切片内容 |
| 后续 | 自选"观察者代号"（如"林中鸟""钟表匠"） | 半匿名，保持诗意感，不走真名社交 |
